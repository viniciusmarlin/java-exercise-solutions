# 📚 Lista de Exercícios - Java

Repositório destinado ao armazenamento das resoluções das listas de exercícios desenvolvidas durante meus estudos de **Lógica de Programação** e **Java**.

O objetivo é registrar minha evolução como programador, praticar conceitos fundamentais e documentar as soluções de acordo com o meu entendimento no momento em que cada exercício foi resolvido.

> **Observação:** As soluções presentes neste repositório não representam necessariamente a melhor implementação possível. Elas refletem meu nível de conhecimento durante o processo de aprendizagem e poderão ser refatoradas futuramente conforme minha evolução.

---


## 📖 Organização

Cada pasta dentro de **Solucoes** corresponde a uma lista de exercícios.

Os arquivos seguem a nomenclatura:

```text
Ex01.java
Ex02.java
Ex03.java
...
```

Cada arquivo contém a resolução de um exercício específico da respectiva lista.
